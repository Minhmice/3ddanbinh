# 3D Asset Pipeline & Performance Optimization

## ⚠️ Sự thật quan trọng nhất

Tốc độ load = 3 thứ:

| # | Yếu tố | Ảnh hưởng |
|---|--------|-----------|
| 1 | **File size** | Network download time |
| 2 | **Decode time** | CPU phải parse geometry/texture |
| 3 | **Render cost** | GPU phải vẽ lên màn hình |

> Đa số chỉ tối ưu file size → nhưng vẫn lag vì 2 cái còn lại.

---

## 📊 Thực trạng hiện tại

| File | Size | Ghi chú |
|------|------|---------|
| `backdrop.glb` | 2 MB | OK |
| `xuong.glb` (Skeleton) | **54 MB** | ❌ Cực nặng |
| `cothuong.glb` (Superficial) | **25 MB** | ❌ Nặng |
| `codetail.glb` (Deep Muscles) | **30 MB** | ❌ Nặng |
| **Tổng** | **~111 MB** | ❌❌❌ |

**Bottleneck hiện tại:**
- Google Drive ≠ CDN (không có edge caching, không có HTTP/2 push)
- Load tất cả 4 model đồng thời = chặn render
- Chưa có compression (Draco/Meshopt)
- Chưa biết texture size trong các GLB

## 🏆 Thứ tự tối ưu (PRIORITY)

### 🥇 Level 1 — BẮT BUỘC

#### 1.1 Draco + Meshopt Compression
```bash
# Draco: nén file size mạnh nhất
npx gltf-pipeline -i model.glb -o model_draco.glb -d

# Meshopt: decode nhanh hơn Draco ở runtime
npx gltfpack -i input.glb -o output.glb
```

> **Insight:** Draco = smaller file, Meshopt = faster runtime.
> Production = dùng **Meshopt** (hoặc BOTH nếu có pipeline).

#### 1.2 Reduce Texture
- 90% lag = **texture** (VRAM killer)
- Mục tiêu: texture ≤ 1K (1024×1024), tối đa 2K cho chi tiết quan trọng
- Pipeline: `PNG → JPG/WebP → KTX2 → GLB`
- Tools: [gltf-transform](https://gltf-transform.dev/), BasisU

#### 1.3 Split Model + Lazy Load
Hiện tại load tất cả 4 layer ngay từ đầu → sai.

```ts
// Chỉ load backdrop + skeleton trước
// Muscle layers load khi user toggle ON
if (user.toggleMuscle) {
  loadModel("muscle.glb")
}
```

→ Giảm initial load từ ~111MB xuống ~56MB (chỉ load backdrop + skeleton).

---

### 🥈 Level 2

#### 2.1 Progressive Loading (như Sketchfab)
```
LOW MODEL → render ngay → HIGH MODEL load sau
```

```ts
loader.load("skeleton_low.glb", (low) => {
  scene.add(low)
  loader.load("skeleton_high.glb", (high) => {
    scene.remove(low)
    scene.add(high)
  })
})
```

→ Perceived performance tăng cực mạnh.

#### 2.2 LOD (Level of Detail)
```
Far → low poly
Near → high poly
```

#### 2.3 Merge Mesh (giảm draw calls)
- Nhiều mesh nhỏ = chết GPU
- Merge mesh trong Blender trước khi export
- Hoặc dùng `THREE.InstancedMesh` cho các phần lặp lại

---

### 🥉 Level 3 — GPU Tuning

#### 3.1 Render on Demand
```ts
if (changed) renderer.render()
```
→ Không render liên tục khi idle.

#### 3.2 Limit Pixel Ratio
```ts
renderer.setPixelRatio(Math.min(window.devicePixelRatio, 1.5))
```
→ Mobile boost cực mạnh (nhiều điện thoại có pixel ratio 3x = render gấp 9 lần pixel).

#### 3.3 Frustum Culling
- Auto trong Three.js, nhưng:
  - Đừng disable
  - Đừng scale object quá lớn

## 🔧 Quy trình đẩy Model lên Web

### Bước 1: Tối ưu model (trong Blender)
1. **Decimate**: Giảm polygon (modifier Decimate) — mục tiêu ≤ 500k triangles tổng
2. **Texture**: Resize về 1K, convert sang JPG/WebP
3. **Clean up**: Tắt Animation, Camera, Lamp khi export
4. **Compression**: Tích Draco khi Export GLB

### Bước 2: Upload lên Google Drive
1. Tải file `.glb` đã tối ưu vào thư mục Drive
2. Đảm bảo quyền: "Anyone with the link can view"
3. Cập nhật `manifest.json` nếu đổi folder:
```json
{
  "folderId": "YOUR_FOLDER_ID",
  "dest": "public/models"
}
```

### Bước 3: Tải về local / deploy
```bash
# Tải model về máy
npm run fetch-models

# Build + deploy (Coolify sẽ tự chạy prebuild → fetch models)
git push
```

## 🎯 Mục tiêu tối ưu

| Metric | Hiện tại | Mục tiêu |
|--------|----------|----------|
| Tổng file size | ~111 MB | ≤ 15 MB |

| First meaningful paint | 10–20s+ (mobile) | ≤ 3s |

## ⚡ Quick Win — Có thể làm ngay KHÔNG cần sửa code

```bash
# 1. Kiểm tra chi tiết từng model
npx gltf-transform inspect public/models/xuong.glb

# 2. Nén Draco
npx gltf-pipeline -i public/models/xuong.glb -o public/models/xuong_opt.glb -d

# 3. Resize texture về 1K
npx gltf-transform resize public/models/xuong.glb public/models/xuong_1k.glb --width 1024 --height 1024

# 4. Full pipeline (Draco + Texture Resize + Quantize)
npx gltf-transform optimize public/models/xuong.glb public/models/xuong_web.glb
```
