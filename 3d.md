# 3D Asset Pipeline & Optimization Guide

Dự án này sử dụng quy trình tự động để quản lý và tải các mô hình 3D từ Google Drive về thư mục `public/models`.

## 1. Quy trình hoạt động (Workflow)

Hệ thống hoạt động dựa trên 3 thành phần chính:
- **`manifest.json`**: Chứa `folderId` của thư mục trên Google Drive.
- **`scripts/fetch-drive-assets.py`**: Script Python sử dụng thư viện `gdown` để quét và tải toàn bộ file trong thư mục Drive đó.
- **`package.json`**: Tự động chạy script tải model trước khi `dev` hoặc `build`.

### Cách đẩy model mới lên web:
1. **Upload lên Drive**: Tải các file `.glb` vào thư mục Google Drive đã cấu hình (hoặc thay đổi `folderId` trong `manifest.json`).
2. **Cấu hình quyền**: Đảm bảo thư mục Drive ở chế độ "Anyone with the link can view".
3. **Cập nhật local/server**: 
   - Chạy `npm run fetch-models` để tải về máy.
   - Khi deploy (Coolify), quá trình `build` sẽ tự động tải bản mới nhất về server.

## 2. Cách giảm dung lượng Model (Optimization)

Để tăng tốc độ load trang (đặc biệt là trên Mobile), các model 3D cần được tối ưu hóa. Dưới đây là các kỹ thuật khuyên dùng:

### A. Sử dụng Draco Compression (Quan trọng nhất)
Draco là thư viện nén hình học mạnh mẽ giúp giảm dung lượng file `.glb` từ 50% - 90%.
- **Cách làm**:
  - Trong Blender: Khi Export `.glb`, tìm phần **Compression** và tích vào **Draco**.
  - Sử dụng CLI: Cài đặt `gltf-pipeline` và chạy:
    ```bash
    npx gltf-pipeline -i model.glb -o model_compressed.glb -d
    ```

### B. Giảm số lượng đa giác (Mesh Simplification)
Số lượng tam giác (Triangles) quá lớn sẽ làm nặng CPU/GPU của điện thoại.
- **Mục tiêu**: Cố gắng giữ tổng toàn bộ các layer dưới **500k triangles**.
- **Cách làm**: Sử dụng modifier **Decimate** trong Blender để giảm số lượng cạnh mà vẫn giữ được form của mô hình.

### C. Tối ưu hóa Texture
Texture thường là nguyên nhân chính gây lag bộ nhớ (VRAM).
- **Kích thước**: Chỉ nên dùng 1024x1024 (1K) hoặc tối đa 2048x2048 (2K) cho các chi tiết cực kỳ sắc nét. Không dùng 4K.
- **Định dạng**: Sử dụng **Basis Universal** hoặc nén về **JPG/WebP** thay vì PNG nếu không cần transparency.

### D. Loại bỏ dữ liệu thừa
Khi export từ Blender, hãy tắt các tùy chọn không cần thiết:
- Tránh export toàn bộ Scene (chỉ chọn Mesh cần thiết).
- Tắt **Animation** nếu layer đó là tĩnh.
- Tắt **Cameras/Lamps**.

## 3. Các công cụ hỗ trợ
- **[gltf.report](https://gltf.report/)**: Kéo thả file `.glb` vào đây để xem chi tiết layer nào nặng nhất, chạy script tối ưu (Simplify, Draco) ngay trên trình duyệt.
- **[Blender](https://www.blender.org/)**: Công cụ mạnh nhất để chỉnh sửa mesh và texture.
- **[RapidCompact](https://rapidcompact.com/)**: Dịch vụ tự động tối ưu hóa model cho web.
